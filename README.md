nodejs-opennebula
=================

Opennebula Cloud API for nodejs


Installation
------------

**Install Node.js:**
```shell
wget -N http://nodejs.org/dist/node-latest.tar.gz
tar xzvf node-latest.tar.gz && cd node-v*
./configure
make install
```
